## Proyecto

Implementación de una pila dinámica en Java aplicada a la gestión de llamadas a rutinas.

El proyecto simula el comportamiento del **call stack** utilizando el principio **LIFO (Last In, First Out)**.

Se almacenan rutinas con:

- Nombre de la rutina  
- Tipo de retorno  

La interacción se realiza mediante un menú en consola que permite agregar, eliminar y visualizar elementos de la pila.

---

## Funcionalidades

- Agregar llamada (Push)
- Mostrar elemento en la cima (Peek)
- Eliminar la cima (Pop)
- Mostrar toda la pila con representación visual ASCII
- Interacción mediante menú en consola
